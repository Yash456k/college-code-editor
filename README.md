# Real-Time Code Editor

A collaborative real-time code editor built with React, Node.js, Express, and Socket.io. This application allows multiple users to join the same "room" and edit code together in real-time.

## Features

- **Real-Time Collaboration**: Changes made by one user are immediately visible to all other users in the same room.
- **Unique Room IDs**: Create or join specific rooms using unique IDs.
- **User Presence**: See a list of all users currently active in the room.
- **Syntax Highlighting**: Powered by CodeMirror for a smooth editing experience.
- **Toast Notifications**: Get notified when users join or leave.

## Tech Stack

- **Frontend**: React, React Router, CodeMirror, Socket.io-client, React Hot Toast
- **Backend**: Node.js, Express, Socket.io
- **Utilities**: UUID, React Avatar

## Prerequisites

Before you begin, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v14 or higher recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)

## Setup

1. Clone the repository
```bash
git clone <repository-url>
```

2. Go into the project folder
```bash
cd realtime-code-editor
```

3. Install dependencies
```bash
npm install
```

4. Start the app (builds the frontend and runs the server)
```bash
npm start
```

Open http://localhost:5000 in your browser.
